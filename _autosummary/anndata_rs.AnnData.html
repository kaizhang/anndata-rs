anndata_rs.AnnData

class anndata_rs.AnnData(*, filename, X, n_obs, n_vars, obs, var, obsm, obsp, varm, varp, uns)

An annotated data matrix. AnnData stores a data matrix X together with annotations of observations obs (obsm, obsp), variables var (varm, varp), and unstructured annotations uns. AnnData is stored as a HDF5 file. Opening/creating an AnnData object does not read data from the HDF5 file. Data is copied to memory only when individual element is requested (or when cache is turned on).
