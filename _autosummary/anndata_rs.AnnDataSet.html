anndata_rs.AnnDataSet

class anndata_rs.AnnDataSet

Similar to AnnData, AnnDataSet contains annotations of observations obs (obsm, obsp), variables var (varm, varp), and unstructured annotations uns. Additionally it provides lazy access to concatenated component AnnData objects, including X, obs, obsm, obsp.

Notes

AnnDataSet doesn’t copy underlying AnnData objects. It stores the references to individual anndata files. If you move the anndata files to a new location, remember to update the anndata file locations when opening an AnnDataSet object.
